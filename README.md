# go-basic
自學Go

Run

```
go run XXXXX.go
```

Example:

```
cd ex-01-hello-world; go run hello.go
```

Build Binary

```
go build -o $dstDir/$execName XXXXX.go
```

Example:

```
go build -o ./bin/hello hello.go
```
